4. Cypher Commands

| Cypher Clause and Syntax | Description |
| ------- | ----- |
| CREATE (n:Label {property1: value1, property2: value2}) | Creates a node with optional labels and properties. |
| MERGE (n:Label {property: value}) | Creates a node if it doesn’t already exist. |
| MATCH (a:Label1), (b:Label2) <br/> CREATE (a)-[r:RELATIONSHIP_TYPE {property: value}]->(b) | Creates a relationship between two nodes. | 
| MATCH (a:Label1), (b:Label2) <br/> MERGE (a)-[:RELATIONSHIP_TYPE]->(b) | Creates a relationship if it doesn’t already exist. |
| MATCH (n:Label {property1: value1}) RETURN n | Finds nodes based on a pattern. |
| MATCH (a:Label1)-[r:RELATIONSHIP_TYPE]->(b:Label2) RETURN a, r, b | Finds nodes and their relationships. |
| MATCH (n:Label {property1: value1}) <br/> SET n.property2 = value2 | Sets or updates properties on a node. |
| MATCH (a:Label1)-[r:RELATIONSHIP_TYPE]->(b:Label2) <br/> SET r.property = value | Sets or updates properties on a relationship. |
| MATCH (n:Label) <br/> RETURN n.property | Retrieves data from the query. |
