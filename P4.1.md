1. Installing the Redis Server using student account

   1.1 Update source.list of Ubuntu by replacing all occurrences of **http** in with **https**.
   ~~~bash
   student@MyPC:~$ sudo nano /etc/apt/sources.list
   ~~~~

   1.2 Install Redis
   ~~~bash
   student@MyPC:~$ sudo apt update
   student@MyPC:~$ sudo apt install -y redis-server
   ~~~~

   1.3 Configure Redis
   
      i. Edit the Redis configuration file
      ~~~bash
      student@MyPC:~$ sudo nano /etc/redis/redis.conf
      ~~~
      > Find the supervised directive and set this to systemd to allow redis to be managed as a service, a.k.a **supervised systemd**.

   3.2 Create a Jupyter notebook and test the redis library
      Read more at URL https://redis.io/docs/latest/develop/connect/clients/python/redis-py/

      ~~~bash
      import redis
      # Create a connection to the Redis server
      r = redis.Redis(host='localhost', port=6379, db=0)
      # Set a key-value pair
      r.set('mykey', 'myvalue')
      # Retrieve the value of the key
      value = r.get('mykey')
      print(value.decode('utf-8'))  # Outputs: myvalue

      # Set multiple key-value pairs
      r.mset({"key1": "value1", "key2": "value2"})
      # Get multiple values using their keys
      values = r.mget(["key1", "key2"])
      for val in values:
         print(val.decode('utf-8'))
      
      # Delete a key-value pair
      r.delete('mykey')
      # Check if a key exists
      exists = r.exists('mykey')
      print(exists)  # Outputs: 

      # To retrieve all existing keys
      key_list = [key.decode('utf-8') for key in r.scan_iter('*')]
      print(key_list)
      # Saving and retrieving Python dictionaries
      
      import json

      # Create Python dictionary as sample data
      data = {
          "name": "Alice",
          "age": 30,
          "email": "alice@example.com"
      }

      # Serialize the dictionary to JSON string
      json_data = json.dumps(data)

      # Store in Redis
      r.set("user:1001", json_data)

      # Retrieve and deserialize
      retrieved_data = json.loads(r.get("user:1001"))
      print(retrieved_data)
      ~~~

2.  Redis CLI Commands

| Command | Description |
| ------- | ----------- |
| SET key value |Stores a string value associated with a key. Overwrites existing value if the key already exists. |
| GET key | Retrieves the value associated with the specified key. |
| APPEND key value | Appends the given value to the existing value of the key. If the key does not exist, it creates the key with the given value. |
| STRLEN key | Returns the length of the string stored at the specified key. |
| RPUSH key value1 [value2 ...] | Appends one or more values to the tail (right side) of a list. Creates the list if it does not exist. |
| LPOP key | Removes and returns the first element (head) of the list stored at the specified key. |
| RPOP key | Removes and returns the last element (tail) of the list stored at the specified key. |
