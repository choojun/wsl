# PRACTICAL 3.1: PySpark Resilient Distributed Datasets (RDDs)

0. Launch the setup Ubuntu-xx.xx distro using PowerShell. Subsequently, log in as the user hduser and start HDFS and YARN services.

1. Login as the user student, and put data files in HDFS

   1.1 Download the sample_data and sample_code folders from the Google Classroom and extract into your D: drive. 

   1.2 Create a project folder:
      > student@MyPC:~$ mkdir de-prj

   1.3 Copy sample data from Windows to WSL:
   
      Copy the entire sample_data folder from Window’s D: drive to the student’s home directory in the local OS in the distro:
      > student@MyPC:~$ cp -r /mnt/c/sample_data /home/student/

   1.4 Copy the sample code from Windows to WSL:
   
      Copy the contents of the sample_code folder from Window’s C:\ drive to the project directory de-prj in the local OS:
      > student@MyPC:~$ cp -r /mnt/c/sample_code/* /home/student/de-prj

2. Put the Sample Data into HDFS

   Put the sample_data folder into HDFS as a directory named data:
   > student@MyPC:~$ hdfs dfs -put sample_data data

3. Set up the Python venv
   
   3.1 Run the script to create a venv:
      > student@MyPC:~$ cd de-prj
      > 
      > student@MyPC:~/de-prj$ ./create_venv.sh

      and enter the following information as prompted:

      - name of the venv (e.g., de-venv)
      - path to the folder where the venv folder will be located  (e.g., /home/student)

   3.2 Navigate to student’s home directory:
      > student@MyPC:~/de-prj$ cd ~
      > 
      > student@MyPC:~$ 


3. Launch JupyterLab

   2.1 Run the JupyterLab service
      > student@MyPC:~$ jupyter lab

   2.2 Access JupyterLab in your browser

4. PySpark RDD

   3.1 Open the Jupyter notebook NOTEBOOK 3.1 Spark RDDs.ipynb in your JupyterLab’s de directory.

   3.2 For each cell, i) review and understand the code, and ii) run the code and observe its output.
   > In file .profile of user student account, you need to export your $PYSPARK_PYTHON with your virtualenv, i.e., export PYSPARK_PYTHON={path/to/your/virtualenv}/bin/python
   > 
   > Example:
   > 
   > export PYSPARK_PYTHON=/home/student/de-prj/de-venv/bin/python
